Overview:
This project provides a comprehensive analysis of football match data from the 2000/2001 season to the 2017/2018 season. It includes team statistics, goals scored and conceded, wins, losses, draws, streaks, and head-to-head comparisons.

Tools & Technologies:

Excel: Data cleaning and preprocessing.

SQL Server: Data storage and queries.

SSIS: ETL processes to load and transform data.

SSAS: Creating measures and dimensions for OLAP analysis.

Power BI: Interactive dashboards and visualizations.

Features:

Analyze team performance across all seasons or a selected season.

Interactive dashboards with slicers for teams and seasons.

Head-to-head analysis between teams.

Stats on wins, losses, draws, goals, clean sheets, and streaks.

Ability to drill down into home and away performances
